# References: Phylogenetics and Evolutionary Graphs

1. [Phylogenetic Tree](https://en.wikipedia.org/wiki/Phylogenetic_tree) - Wikipedia - Explains phylogenetic tree concepts including rooted and unrooted trees, cladograms, phylograms, branch lengths, and methods for reading and interpreting evolutionary relationships.

2. [Molecular Clock](https://en.wikipedia.org/wiki/Molecular_clock) - Wikipedia - Describes the molecular clock hypothesis for estimating divergence times from sequence data, covering neutral theory, rate calibration, and relaxed clock models in evolutionary analysis.

3. [Maximum Parsimony (Phylogenetics)](https://en.wikipedia.org/wiki/Maximum_parsimony_(phylogenetics)) - Wikipedia - Overview of the parsimony criterion for phylogenetic inference, explaining tree scoring, branch-and-bound search, and comparison with maximum likelihood and Bayesian methods.

4. Molecular Evolution: A Phylogenetic Approach - Roderic Page - Blackwell Science - Covers phylogenetic tree construction methods, distance matrices, character-based methods, bootstrapping, and molecular clock analysis with biological examples throughout.

5. Inferring Phylogenies - Joseph Felsenstein - Sinauer Associates - The definitive reference on phylogenetic methods including parsimony, likelihood, Bayesian inference, consensus trees, and network approaches for reticulate evolution.

6. [MEGA: Molecular Evolutionary Genetics Analysis](https://www.megasoftware.net/) - MEGA Team - Documentation for the widely used MEGA software for sequence alignment, phylogenetic tree construction, molecular clock testing, and evolutionary distance estimation.

7. [Interactive Tree of Life (iTOL)](https://itol.embl.de/help.cgi) - EMBL - Help documentation for the iTOL web tool for displaying, annotating, and managing phylogenetic trees, supporting Newick, Nexus, and PhyloXML input formats.

8. [PhyloT: Phylogenetic Tree Generator](https://phylot.biobyte.de/) - biobyte solutions - Web tool for generating phylogenetic trees from NCBI taxonomy, useful for exploring taxonomic relationships and understanding tree topology before computational reconstruction.

9. [ETE Toolkit Documentation](http://etetoolkit.org/docs/latest/) - ETE Toolkit - Python framework for phylogenetic tree analysis and visualization, covering tree manipulation, comparison algorithms, and publication-quality rendering of evolutionary graphs.

10. [Phylogenetic Network Wiki](https://phylnet.info/) - Phylogenetic Networks - Resource on phylogenetic networks that represent reticulate evolution including hybridization, horizontal gene transfer, and recombination events not captured by tree models.
