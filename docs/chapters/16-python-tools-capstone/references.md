# References: Python Tools and Capstone Projects

1. [Biopython](https://en.wikipedia.org/wiki/Biopython) - Wikipedia - Overview of the Biopython library for computational biology, covering modules for sequence analysis, file parsing, database access, phylogenetics, and structural bioinformatics in Python.

2. [NetworkX](https://en.wikipedia.org/wiki/NetworkX) - Wikipedia - Describes the Python library for creating, manipulating, and analyzing complex networks, covering graph data structures, algorithms, and visualization capabilities used throughout bioinformatics.

3. [Jupyter Notebook](https://en.wikipedia.org/wiki/Project_Jupyter) - Wikipedia - Explains the Jupyter ecosystem for interactive computing, covering notebook format, kernel architecture, and how notebooks enable reproducible bioinformatics analysis with embedded code, text, and visualizations.

4. Python for Biologists - Martin Jones - Self-published - Practical introduction to Python programming for biologists covering string manipulation, file parsing, regular expressions, and biological data processing with clear examples.

5. Bioinformatics with Python Cookbook (3rd Edition) - Tiago Antao - Packt Publishing - Recipe-based guide to bioinformatics in Python covering Biopython, genomics pipelines, population genetics, phylogenetics, and network analysis with reproducible code examples.

6. [Biopython Tutorial and Cookbook](https://biopython.org/wiki/Documentation) - Biopython - Official documentation and tutorial covering sequence objects, file parsing, BLAST integration, GenBank access, phylogenetics, and structural biology modules for Python-based bioinformatics.

7. [NetworkX Tutorial](https://networkx.org/documentation/stable/tutorial.html) - NetworkX Developers - Official tutorial for creating and analyzing graphs in Python, covering graph construction, node and edge attributes, algorithms, and drawing functions essential for biological network analysis.

8. [Neo4j Python Driver Documentation](https://neo4j.com/docs/python-manual/current/) - Neo4j Inc. - Guide to connecting Python applications to Neo4j graph databases, covering session management, Cypher query execution, and transaction handling for biological knowledge graph applications.

9. [Snakemake Documentation](https://snakemake.readthedocs.io/en/stable/) - Snakemake - Documentation for the Python-based workflow management system, covering rule definitions, dependency resolution, and cluster execution for reproducible bioinformatics pipeline development.

10. [Pandas Documentation](https://pandas.pydata.org/docs/getting_started/index.html) - pandas Development Team - Getting started guide for the Python data analysis library, covering DataFrames, data import and export, merging, and grouping operations fundamental to bioinformatics data processing.
