# References: Protein-Protein Interaction Networks

1. [Protein-Protein Interaction](https://en.wikipedia.org/wiki/Protein%E2%80%93protein_interaction) - Wikipedia - Overview of physical and functional protein interactions, detection methods including yeast two-hybrid and co-immunoprecipitation, and the biological significance of interaction networks.

2. [Interactome](https://en.wikipedia.org/wiki/Interactome) - Wikipedia - Describes the complete set of molecular interactions in a cell, covering protein interaction maps, experimental and computational interactome mapping, and network-level properties.

3. [Network Medicine](https://en.wikipedia.org/wiki/Network_medicine) - Wikipedia - Explains how network analysis of protein interactions reveals disease modules, drug targets, and comorbidity relationships, bridging molecular biology and systems-level clinical understanding.

4. Networks in Cell Biology - Mark Buchanan - Cambridge University Press - Accessible introduction to biological network analysis covering PPI network topology, hubs, modules, and the relationship between network structure and cellular function.

5. Network Biology: Understanding the Cell's Functional Organization - Albert-Laszlo Barabasi - Nature Reviews Genetics - Landmark review establishing network biology principles including scale-free topology, network motifs, and the centrality-lethality rule in protein interaction networks.

6. [STRING Database Help](https://string-db.org/cgi/help) - STRING Consortium - Documentation for the STRING protein interaction database, explaining interaction scores, evidence channels, network visualization, and API access for computational analysis.

7. [BioGRID Documentation](https://wiki.thebiogrid.org/) - BioGRID Project - User guide for the Biological General Repository for Interaction Datasets, covering curated interaction data, experimental evidence codes, and bulk download options.

8. [Cytoscape User Manual](https://manual.cytoscape.org/) - Cytoscape Consortium - Documentation for the Cytoscape network visualization platform, covering network import, layout algorithms, visual styles, and app ecosystem for biological network analysis.

9. [IntAct Molecular Interaction Database](https://www.ebi.ac.uk/intact/documentation) - EMBL-EBI - Documentation for the IntAct database of experimentally validated molecular interactions, explaining curation standards, MI ontology, and PSI-MI data exchange formats.

10. [NetworkX Graph Algorithms](https://networkx.org/documentation/stable/reference/algorithms/index.html) - NetworkX Developers - Reference for Python graph algorithms applicable to PPI networks, including centrality measures, community detection, shortest paths, and network motif analysis.
